Unikta

ᎤᏂᎩᏔ/Unikta is a Cherokee font and character encoder for Playdate Pulp, which lets people write videogames and interactive stories without using any code. This tool lets you type in Cherokee syllabary on the left, and get text to paste into Pulp on the right.

You can download Unikta and learn more about how to use it here!
